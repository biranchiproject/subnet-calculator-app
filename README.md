# Subnet Calculator App - brog Creativity

A modern, professional IP Subnet Calculator designed for network architects and IT professionals. Built with React, Tailwind CSS, and Framer Motion, featuring a high-fidelity neon cyber aesthetic.

## Features

- **IPv4 Analysis**: Support for Class-based (A, B, C) and CIDR calculations.
- **IPv6 Gateway**: Next-generation prefix analysis with massive address space insights.
- **Neon UI**: Immersive dark theme with pulsating neon glows and glassmorphism.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.
- **RFC Compliant**: Accurate logic engine following standard networking protocols.

## Tech Stack

- **React 19**
- **Tailwind CSS 4**
- **React Router 7**
- **Framer Motion**
- **Lucide Icons**
- **Vite**

## Quick Start

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

## Deployment

This project is ready for **Cloudflare Pages**.
1. Connect this repository to your Cloudflare Dashboard.
2. Set Build Command to `npm run build`.
3. Set Build Output Directory to `dist`.
